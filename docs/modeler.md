## Download

Download modeler from the [camunda official resource](https://downloads.camunda.cloud/release/camunda-modeler/3.7.3/camunda-modeler-3.7.3-linux-x64.tar.gz
) 
 
## Install and run

Unpack to any folder and run the binary file

````
./camunda-modeler
````
